# Chanho's dev blog

[![Netlify Status](https://api.netlify.com/api/v1/badges/d52613d2-028c-4166-bd14-b7784176e05e/deploy-status)](https://app.netlify.com/projects/chanho-dev-blog/deploys)

This project is created by [Next.js](https://nextjs.org), deployed on [Netlify](https://www.netlify.com).

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
# http://localhost:3000
pnpm dev
```

If you want to run both frontend and netlify functions, use:

```bash
# http://localhost:8888
netlify dev
```
